#Git
- Look for the "Terminal" command prompt application if you're using a Mac.
- Open "Git Bash" or the Windows command prompt if you're using a Windows computer.
- Type git version once your terminal program has launched. The output will either reveal the installed version of Git or warn you that git is a command you are unfamiliar with. If the command is unknown, continue reading to learn how to install Git.
1. Locate the most recent Git for Windows installation and download it.
2. Once the installer has begun, proceed as directed on the Git Setup wizard screen to finish the installation.
3. Launch Windows Command Prompt (or Git Bash, if you decided against using the default Git Windows Command Prompt during the Git installation).
